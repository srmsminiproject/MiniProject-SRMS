What is the difference between Stack and Queue?

The main difference between a stack and a queue is the order in which data is added and removed. A stack uses a last-in, first-out (LIFO) model, while a queue uses a first-in, first-out (FIFO) model.

What is Big-O notation in algorithms?

Big O notation is a way to describe how the time or space needed by an algorithm grows as the input size increases. It's used in computer science, mathematics, and complexity theory.
